# GitHub Repo Analyzer

## Project Description
This is a Streamlit-based web application that analyzes GitHub repository statistics using GitHub REST API.

## Features
- Displays stars
- Shows forks
- Displays watchers
- Shows open issues
- Displays repository size
- Shows creation and last updated date

## Technologies Used
- Python
- Streamlit
- GitHub REST API
- Requests Library

## Architecture
User Input → Streamlit UI → GitHub API → JSON Response → Display Stats

## How to Run Locally
1. Install Python
2. Install dependencies:
   pip install streamlit requests
3. Run:
   streamlit run app.py

## Live Application
Deployed using Streamlit Cloud.
